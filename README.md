# DSAV — DSA Algorithms Visualizer

> 🚧 **Currently under active development.**

An interactive visualizer for Data Structures & Algorithms, built to make learning DSA intuitive and visual.

🔗 **Live Demo:** [dsa-algorithms-visualizer-jjk2.vercel.app](https://dsa-algorithms-visualizer-jjk2.vercel.app/)

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | React 18 + TypeScript |
| Build Tool | Vite |
| Styling | Tailwind CSS |
| Animations | Framer Motion |
| Icons | Lucide React |
| Deployment | Vercel |

---

## What's Implemented

### Sorting Algorithms
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

### Searching Algorithms
- Linear Search
- Binary Search

### Data Structures
- Stack (LIFO)
- Queue (FIFO)
- Binary Tree — Create & Visualize
